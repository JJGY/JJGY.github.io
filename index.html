About Me

Data Science Projects

Time Series Prediction of Outbreak of Infectious Diseases

This project explores time series and deep learning models—SMA, ARMA, RNN, and LSTM—to predict infectious disease outbreaks using digital data such as Google search trends. Using data from states like New York and California, models were trained to forecast COVID-19 cases for early detection.

Impact of COVID on Education: A Data-Driven Analysis

This project uses regression models (Linear, Lasso, k-NN, Decision Trees, Bagging, Random Forest, Adaboost) to estimate how COVID-19 impacted global primary education rates using economic indicators. Adaboost achieved top accuracy (R² = 0.81).
